# `closingBracketSameLine`

Control the closing bracket (`>`) of a multi-line element should come at the end of the last line
or on the next line (with a line break).

Default option is `false`.

## Example for `false`

```html
<span
  class=""
  style=""
></span>
```

## Example for `true`

```html
<span
  class=""
  style=""></span>
```
